Xây dựng ứng dụng Todo List đơn giản bằng HTML, CSS và JavaScript

Giới thiệu

Todo List là một ứng dụng giúp bạn ghi chú những việc cần làm. Trong bài viết này, bạn sẽ học cách xây dựng một ứng dụng đơn giản với 3 chức năng cơ bản: thêm, hiển thị, xóa công việc. Không cần framework, chỉ dùng HTML, CSS, JavaScript thuần.

Bước 1: Tạo cấu trúc HTML

Chúng ta cần một giao diện đơn giản: ô nhập công việc, nút thêm và danh sách công việc. 

        
          <input type="text" id="todo-input" placeholder="Nhập công việc..." /><button id="add-btn">Thêm</button><ul id="todo-list"></ul>

Bước 2: Thiết kế giao diện bằng CSS

        
          body {
            font-family: Arial;
            background: #f2f2f2;
           }

          .todo-container {
          background: white;
          padding: 30px;
          border-radius: 10px;
          }

Bước 3: Thêm chức năng bằng JavaScript

        
          addBtn.addEventListener("click", () => {
          const text = input.value.trim();
          if (text !== "") {
          const li = document.createElement("li");
          li.innerHTML = `<span>${text}</span><button class="delete-btn">X</button>`;
          list.appendChild(li);
          input.value = "";
            }
          });

          list.addEventListener("click", function (e) {
            if (e.target.classList.contains("delete-btn")) {
              const li = e.target.parentElement;
              list.removeChild(li);
            }
          });

Bonus: Thêm công việc bằng phím Enter

        
          input.addEventListener("keypress", function (e) {
          if (e.key === "Enter") {
          addBtn.click();
            }
          });

Tổng kết

  • Tương tác DOM bằng JavaScript
  • Giao diện cơ bản với CSS
  • Tư duy xử lý dữ liệu người dùng
